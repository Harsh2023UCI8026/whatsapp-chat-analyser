# whatsapp-chat-analyser
A web-based application built using Streamlit that analyzes exported WhatsApp chat files and provides detailed insights such as message statistics, timelines, activity maps, word clouds, and emoji analysis.

Features:
1.Total messages, words, media & links count 
2.Monthly & daily chat timeline  
3.Most active days & months
4.Weekly activity heatmap
5.Most active users in group chats  
6.Word cloud visualization  
7.Most common words analysis  
8.Emoji usage analysis  
User-wise or overall chat analysis 


Tech Stack:
1.Language: Python 
2.Framework: Streamlit  
3.Libraries: Pandas, Matplotlib, Seaborn, Regex, WordCloud
